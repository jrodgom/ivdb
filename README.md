# IVDb - Base de Datos de Videojuegos

Proyecto web para gestión y valoración de videojuegos.

## Tecnologías

- React + Vite
- Django REST Framework
- TailwindCSS
- RAWG API

## Configuración

1. Copiar `.env.example` a `.env` y configurar variables
2. Backend: `python manage.py runserver`
3. Frontend: `npm install && npm run dev`

## Características

- Búsqueda de juegos
- Sistema de valoraciones
- Favoritos
- Perfiles de usuario
